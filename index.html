Regex Function Collection

There are a few useful functions that deal with regex in C++ and you can use them in a convenient way. They have been written with std::regex; so you no need to worry about linkage problem and so forth.

Just you need: #include "rfc/rfc.hpp", of course after cloning

So: git clone https://github.com/k-five/RFC/tree/master/cpp

Then: cd cpp

And inside cpp, you will see rfc and inside that directory, you can find all header files. You can use them all or just one header file or even copy any function you like into your source code.

Since there are several bugs related to this library, you should be aware fo using which compiler version. And the last bug that I could found is on versions g++ 6.3.0 and lower. Therefore you should always use the least version, that it currently is g++ 7.1.0.

More detail about this bug

Also you can read about another bug related to negative-lookahead-assertion here on stackoverflow.com in versions g++ 5.3.0 and lower.
