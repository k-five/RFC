Regex Function Collection

Some useful functions to deal with Regular Expressions

All written by standard library.

Now, first version of `C++` is available.


It has:

  1. valid family
  2. parse family
  3. split family
  4. substitute family
  5. join family
  6. match family
  7. count family

They are just hpp files and you no need to use more than a simple include: #include <rfc/rfc.hpp>

Since one of the most buggy library is std::regex you should be aware of updating you compiler to the least version
